# Toko Bunga Aurora — Project

Situs demo untuk toko bunga dengan tampilan elegan dan interaksi dasar.

File penting:
- [index.html](index.html) — halaman utama
- [styles.css](styles.css) — styling
- [script.js](script.js) — interaksi (render produk, keranjang, modal)

Cara mencoba cepat:
1. Buka `index.html` di browser modern.
2. Klik "Lihat Koleksi" atau scroll ke bagian produk.

Catatan:
- Gambar diambil dari Unsplash via URL; untuk produksi, simpan aset lokal.
- Fitur checkout hanyalah demo (alert).
